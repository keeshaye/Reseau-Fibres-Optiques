# Reseau-Fibres-Optiques
Ce projet vise la reconstitution et la réorganisation d’un réseau de fibres optiques à partir de chaînes décrivant les connexions entre clients. Le réseau est modélisé sous forme de graphe et reconstruit à l’aide de structures de données telles que liste chaînée,table de hachage et arbre quaternaire, afin d’analyser les performances algorithmiques.
